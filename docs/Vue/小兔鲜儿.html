  1. 说说小兔仙项目中的登录是如何实现的 (登录流程是怎么样的, 采用了哪种技术实现的身份验证 jwt)
 使用身份校验jwt 全称是json web token 
 简洁可以在get post参数 发送 数据量小 传输速度也很快
 自包含：负载中包含了所有用户所需要的信息 避免了查询数据库
 Token是以JSON加密的形式保存在客户端中，由于JWT是跨语言的 原则上任何web心事形式都支持
 不需要再服务端保存会话信息 适用于分布式服务
比较是安全
